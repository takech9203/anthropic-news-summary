VS Code /usage でキャッシュミス、ロングコンテキスト、サブエージェント、MCP ごとの内訳を表示

バックグラウンドセッションが起動元シェルの ANTHROPIC_* 変数を継承しない

Before / After

Before

GovCloud リージョンでプレフィックスが global として導出 → 400 エラー発生

After

GovCloud リージョンでプレフィックスが正しく us-gov として導出 → 正常に動作

コード例

Bash アップデートと設定確認 
# Claude Code をアップデート
claude update

# GovCloud リージョンが正しく設定されていることを確認
echo $AWS_REGION
# 出力例: us-gov-west-1

# wheelScrollAccelerationEnabled の設定
# settings.json に追加:
# { "wheelScrollAccelerationEnabled": false }
