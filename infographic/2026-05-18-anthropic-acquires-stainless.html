Anthropic が Stainless を買収

2026-05-18

概要

🤝

SDK 生成・MCP サーバーツーリング企業の買収

Anthropic が SDK 自動生成と MCP サーバーツーリングを専門とする Stainless 社を買収。エージェント時代のインフラ基盤を強化する戦略的な動き。

🔧

全 Anthropic 公式 SDK の生成を担当していたチームの統合

Stainless は既に Anthropic の全公式 SDK を生成しており、そのチームが Anthropic に完全統合されることで開発サイクルの高速化が実現。

🤖

エージェント時代のシステム接続性を強化

AI エージェントがあらゆるシステムと連携するために必要なインフラを内製化。MCP を通じたツール接続のエコシステムを拡大。

🌐

TypeScript、Python、Go、Java、Kotlin SDK への影響

5 つの言語で提供される公式 SDK の品質向上と、新機能への迅速な対応が期待される。開発者向けインターフェースに変更なし。

🔗

MCP エコシステムの深化

Model Context Protocol のサーバー生成ツーリングを取り込むことで、MCP エコシステム全体の成熟と標準化を加速。

主要機能・変更点

👥 チームの完全統合

Stainless のエンジニアリングチームが Anthropic に加わり、SDK 開発と API プラットフォームの専門知識を直接活用可能に。

⚙️ SDK 生成の内製化

API 仕様から自動的に高品質な SDK を生成するパイプラインを社内に統合。新 API 機能のリリースと SDK 更新のタイムラグを最小化。

🔗 MCP エコシステムの強化

MCP サーバー生成ツーリングの内製化により、エージェントが接続できるツールやサービスの範囲を大幅に拡大。

✨ 開発者体験の向上

SDK の型安全性、ドキュメント、エラーハンドリングの品質を継続的に改善し、開発者にとって最高のプラットフォーム体験を提供。

リーダーシップの声

Stainless のチームは、開発者が Claude と連携するための基盤を構築してきました。彼らを社内に迎えることで、エージェントの時代に必要なインフラをより迅速に構築できるようになります。

Katelyn Lesse - Anthropic, Head of Developer Platform

Alex Rattray - Stainless, CEO & Co-founder

コードサンプル

SDK のインターフェースに変更はありません。既存のコードはそのまま動作します。

Python Claude API 呼び出し例 
import anthropic

client = anthropic.Anthropic()
message = client.messages.create(
    model="claude-sonnet-4-6-20260318",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hello, Claude!"}]
)
print(message.content[0].text)
TypeScript Claude API 呼び出し例 
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic();
const message = await client.messages.create({
    model: "claude-sonnet-4-6-20260318",
    max_tokens: 1024,
    messages: [{ role: "user", content: "Hello, Claude!" }],
});
console.log(message.content[0].text);
