概要

🔄

/resume でバックグラウンドセッションを統合管理

インタラクティブセッションとバックグラウンドセッションを一覧で確認し、シームレスに切り替え可能に

⚡

起動時の最大 75 秒ハングを修正

MCP サーバー起動のタイムアウトを 15 秒に制限し、高速な起動を実現

🔌

MCP サーバーのページネーション対応修正

ツールリストが 1 ページ目のみ取得される問題を修正し、全ページを正しく取得

🎨

ターミナルレンダリングの安定性向上

リサイズ時のクラッシュ、長時間セッションでの表示崩れ、スピナーの改善

🛠️

プラグインマーケットプレイスの UX 改善

HTTPS 優先設定、更新日表示、検索性の向上でプラグイン管理がより便利に

新機能

🔄 /resume バックグラウンドセッション対応

バックグラウンドで実行中のセッションも /resume 一覧に表示。bg マーク付きで状態を確認可能

⏱️ 完了通知に経過時間

バックグラウンドサブエージェントの完了通知に実行時間を表示し、タスクの所要時間を把握

🎯 /model セッションスコープ化

/model コマンドでセッション単位のモデル変更が可能に。'd' キーでデフォルト設定に戻す

📁 /bg デタッチ時ディレクトリ保持

バックグラウンドにデタッチする際、作業ディレクトリを正しく維持

🔒 CLAUDE_CODE_PLUGIN_PREFER_HTTPS 対応

環境変数でプラグインの HTTPS 通信を優先設定可能に

📅 /plugin 更新日表示

プラグイン一覧に最終更新日を表示し、メンテナンス状況を可視化

Before / After

Before

  • /resume にバックグラウンドセッション表示なし
  • 起動時に最大 75 秒ハング
  • MCP ツールがページ 1 のみ取得

After

  • /resume で bg マーク付き統合一覧
  • 15 秒タイムアウトで高速起動
  • 全ページを正しく取得

コードサンプル

Bash /resume と /model の使用例 
# バックグラウンドでタスクを開始
claude --bg "プロジェクト全体のテストを実行して結果をまとめてください"

# /resume でセッション一覧を表示 (bg マーク付き)
> /resume
#   1. [bg] プロジェクト全体のテストを実行... (running · 3m 22s)
#   2. [bg] 依存パッケージの脆弱性スキャン... (completed · 1m 45s)
#   3.      前回のインタラクティブセッション (paused · 2h ago)

# /model でセッションスコープのモデル変更
> /model
# 'd' キーでデフォルト設定、Enter で現セッションのみ変更
