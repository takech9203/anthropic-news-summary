/doctor CLAUDE.md トリミングフロー

flowchart TD
    subgraph Input["入力"]
        MD["CLAUDE.md"]
        CB["コードベース"]
    end
    subgraph Analysis["分析エンジン"]
        Parse["セクション分析"]
        Derive["推論可能情報の特定"]
        Compare["重複・冗長性判定"]
    end
    subgraph Output["トリミング提案"]
        Keep["保持推奨"]
        Trim["トリミング候補"]
    end
    MD --> Parse
    CB --> Derive
    Parse --> Compare
    Derive --> Compare
    Compare --> Keep
    Compare --> Trim

重要な修正 - Top 5

  • MCP サーバー request_timeout_ms 修正 - タイムアウト設定が正しく適用されるよう修正
  • CLAUDE_CODE_EXTRA_BODY バックグラウンド対応 - バックグラウンドエージェントでも追加ボディパラメータが有効に
  • 期限切れログインのエラーメッセージ改善 - より分かりやすいエラーメッセージで再認証を案内
  • OAuth MCP 再認証修正 - OAuth トークン失効時の再認証フローを修正
  • Bedrock 起動ハング修正 - AWS Bedrock 経由での起動時にハングする問題を解消
