概要

⭐

Opus 4.7 xhigh エフォートレベル

Claude Opus 4.7 専用の新しいエフォートレベル xhigh を追加。high と max の間に位置し、速度とインテリジェンスの最適なバランスを実現します。/effort コマンドのインタラクティブスライダーから選択可能です。

🤖

Auto モードの簡素化

--enable-auto-mode フラグが不要に。Max サブスクライバーが Opus 4.7 使用時にデフォルトで Auto モードが利用可能。v2.1.112 でホットフィックスも適用済みです。

🔍

/ultrareview クラウドコードレビュー

クラウド上で並列マルチエージェント分析と批評を実行する包括的なコードレビュー機能。現在のブランチまたは GitHub PR を対象にセキュリティ、パフォーマンス、アーキテクチャの観点からレビューします。

🛡️

/less-permission-prompts スキル

トランスクリプトをスキャンして頻繁に使用する読み取り専用コマンドを特定し、.claude/settings.json に許可リストを提案。パーミッションプロンプトの頻度を大幅に削減します。

💻

Windows PowerShell ツール

Windows で PowerShell ツールが段階的にロールアウト。CLAUDE_CODE_USE_POWERSHELL_TOOL 環境変数でオプトイン/オプトアウトが可能。Windows ネイティブ環境のサポートが向上しました。

技術的な詳細

⭐ xhigh エフォートシステム

Claude Code のエフォートレベルシステムに xhigh が追加されました。Opus 4.7 専用で、既存の 4 段階に加わる 5 段階目のレベルです。max ほどのレイテンシを発生させずに高いインテリジェンスを発揮します。Opus 4.7 以外で xhigh を指定した場合は自動的に high にフォールバックします。

low → medium → high → xhigh (Opus 4.7) → max

🔍 /ultrareview アーキテクチャ

/ultrareview はクラウドベースのコードレビュー機能です。ローカルの単一エージェントによるレビューとは異なり、クラウド上で複数のエージェントが並列に分析と批評を実行します。引数なしで現在のブランチ、/ultrareview <PR#> で特定の GitHub PR をレビュー。セキュリティ、パフォーマンス、アーキテクチャ、コード品質などの複数の観点から同時に分析が行われます。

🛡️ /less-permission-prompts ワークフロー

過去のトランスクリプトを分析してユーザーが頻繁に許可している読み取り専用の Bash コマンドや MCP ツール呼び出しを特定し、.claude/settings.json に追加する許可リストを提案します。glob パターンを含む読み取り専用 Bash コマンド (例: ls *.ts) と cd <project-dir> && で始まるコマンドはデフォルトでパーミッション不要になりました。

💻 Windows PowerShell ツール

Windows で段階的に自動有効化され、オプトアウトは CLAUDE_CODE_USE_POWERSHELL_TOOL=0 で可能。Linux/macOS では明示的に有効化が必要で、PATH に pwsh が必要です。Windows ユーザーはネイティブの PowerShell 環境でツール実行が可能になり、Bash エミュレーションに依存する必要がなくなります。

📊 OpenTelemetry デバッグ拡張

OTEL_LOG_RAW_API_BODIES 環境変数で完全な API リクエスト/レスポンスボディを OpenTelemetry ログイベントとして出力。前バージョンの TRACEPARENT/TRACESTATE 分散トレースリンキングと組み合わせて CI/CD パイプラインでの詳細なデバッグが可能です。

Auto モードフラグの削除

v2.1.111 で --enable-auto-mode フラグが不要になりました。スクリプトやエイリアスで使用している場合は削除を推奨します。

Before (v2.1.110 以前)

claude --enable-auto-mode "タスクを実行してください"

After (v2.1.111 以降)

claude "タスクを実行してください"
# Auto モードは Max + Opus 4.7 で自動有効

Ctrl+U の動作変更

Before (v2.1.110 以前)

Ctrl+U: カーソル位置から行頭までを削除

After (v2.1.111 以降)

Ctrl+U: 入力バッファ全体をクリア
Ctrl+Y: クリアした内容を復元

PowerShell ツールの制御

Bash PowerShell オプトイン/オプトアウト 
# Windows: オプトアウト
export CLAUDE_CODE_USE_POWERSHELL_TOOL=0

# Linux / macOS: 明示的にオプトイン (pwsh が PATH に必要)
export CLAUDE_CODE_USE_POWERSHELL_TOOL=1
