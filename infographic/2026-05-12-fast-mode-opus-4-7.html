概要

⚡

Fast mode が Opus 4.7 に対応

Research preview として提供されていた Fast mode が、最新の Claude Opus 4.7 モデルをサポート

🔧

有効化方法

speed: "fast" パラメータと fast-mode-2026-02-01 ベータヘッダーで有効化

🚀

最大 2.5 倍の高速化

出力トークン生成速度 (OTPS) が最大 2.5 倍に向上。同一モデルウェイトを使用

💰

料金体系

標準 Opus の 6 倍の料金: $30/MTok 入力、$150/MTok 出力。キャッシュやデータレジデンシーの乗数と併用可能

📅

段階的な拡張

2026 年 2 月に Opus 4.6 向けに初リリース。今回 Opus 4.7 にサポートを拡張

主な機能

🏎️ speed パラメータ

Messages API に speed: "fast" を指定して Fast mode を有効化。シンプルなインターフェースで高速推論にアクセス

📦 ベータヘッダー

fast-mode-2026-02-01 をベータヘッダーに含めることで機能を有効化。SDK の betas パラメータで指定

📊 専用レート制限

Fast mode 専用のレート制限が設定。標準モードとは独立して管理。レスポンスヘッダーで確認可能

💬 速度フィールド

レスポンスの usage.speed フィールドで、実際に Fast mode が適用されたか確認可能

コードサンプル

Python Fast mode 基本実装 
import anthropic

client = anthropic.Anthropic()

response = client.beta.messages.create(
    model="claude-opus-4-7",
    max_tokens=4096,
    speed="fast",
    betas=["fast-mode-2026-02-01"],
    messages=[
        {"role": "user", "content": "Refactor this module to use dependency injection"}
    ],
)

print(response.content[0].text)
Python Fast mode フォールバックパターン 
def create_message_with_fast_fallback(max_retries=None, max_attempts=3, **params):
    try:
        return client.beta.messages.create(**params, max_retries=max_retries)
    except anthropic.RateLimitError:
        if params.get("speed") == "fast":
            del params["speed"]
            return create_message_with_fast_fallback(**params)
        raise

制限事項

  • ⚠️ speed パラメータを切り替えるとキャッシュが無効化される
  • ⚠️ Batch API では利用不可
  • ⚠️ Priority Tier では利用不可
  • ⚠️ Claude Platform on AWS では利用不可
