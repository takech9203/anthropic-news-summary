コードサンプル

Python Managed Agent としてピッチビルダーを起動 
from anthropic import Anthropic

client = Anthropic()

# Managed Agent としてピッチビルダーを起動
agent = client.agents.create(
    name="pitch-builder",
    description="ピッチブック作成エージェント",
    tools=[
        {"type": "mcp", "server": "factset"},
        {"type": "mcp", "server": "sp-capital-iq"},
        {"type": "microsoft365", "apps": ["excel", "powerpoint"]},
    ],
    instructions="クライアントミーティング用のピッチブックを作成してください。"
)

# Dispatch でタスクを割り当て
task = client.agents.dispatch(
    agent_id=agent.id,
    message="来週の ABC Corp とのミーティング用にピッチブックを準備してください。"
)

ユースケース

投資銀行 - クライアントミーティング準備

Pitch Builder と Meeting Preparer を組み合わせ、FactSet や S&P Capital IQ からデータを取得。ピッチブックの作成からクライアントブリーフの準備まで一貫して自動化。

資産運用 - 月次決算プロセス

Month-end Closer が決算チェックリストを実行し、General Ledger Reconciler が NAV 計算を自動化。Statement Auditor が最終レビューを実施し、監査対応準備を完了。

コンプライアンス - KYC・スクリーニング

KYC Screener が Dun & Bradstreet のデータを活用してエンティティの検証を自動化。ソースドキュメントのレビューとエスカレーションパッケージの作成を効率化。
