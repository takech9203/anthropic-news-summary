Computer Use が Cowork と Claude Code で利用可能に + Dispatch の改善

2026-03-23

概要

🖥️

Computer Use が Cowork で利用可能に

Claude がユーザーのコンピュータ画面を直接操作できる Computer Use 機能が Cowork および Claude Code で利用可能になりました。macOS の Claude Desktop アプリにおいて、Pro および Max プランのユーザーが研究プレビューとして利用できます。

🔀

ツール選択の階層構造

Cowork はコネクタ、ブラウザ、画面操作の 3 層構造で最適なツールを自動選択します。直接統合が最も高速かつ正確であり、Computer Use はフォールバックとして位置づけられています。

🚀

Dispatch の改善

スマートフォンからタスクを割り当て、ユーザーが離席中でも Claude がコンピュータを操作してタスクを実行できるようになりました。デバイス間で 1 つの継続的な会話を維持できます。

🛡️

安全対策

各アプリケーションへのアクセス前にユーザー許可を要求し、プロンプトインジェクション検出の自動スキャンを実施。株式取引や機密データ入力の回避などのセーフガードが実装されています。

Before / After

Before

  • コネクタやブラウザ操作のみ利用可能
  • コネクタが存在しないアプリは操作不可
  • Computer Use は API 経由のみ
  • Dispatch からの画面操作に非対応

After

  • コネクタ + ブラウザ + Computer Use の 3 層構造
  • 任意のデスクトップアプリを操作可能
  • Cowork / Claude Code から直接利用
  • Dispatch 経由でリモートから画面操作

主な機能

🖥️ Computer Use

Claude がデスクトップ画面上でクリック、タイピング、ナビゲーションを実行。スクリーンショットベースで画面の状態を把握し、任意のアプリケーションを操作できます。

🔀 ツール自動選択

コネクタ、ブラウザ、画面操作の 3 層から最も精度の高いツールを自動選択。直接統合を優先し、Computer Use はフォールバックとして効率を最適化します。

🚀 Dispatch 改善

スマートフォンからタスクを割り当て、デスクトップ上で Claude が自動実行。デバイス間で 1 つの継続的な会話を維持できます。

🔒 パーミッションモデル

各アプリケーションへのアクセス前にユーザー許可を要求。一部のアプリはデフォルトでアクセス制限されます。

  • 株式取引の回避
  • 機密データ入力の回避
  • 顔画像収集の回避

🛡️ 安全対策

プロンプトインジェクション検出の自動スキャンを実施。画面上のテキストや画像による意図しない動作誘導を防止します。ユーザーはいつでも操作を停止可能です。

ツール選択の階層構造

🔌 1. コネクタ (最優先)

Gmail、Google Drive、Slack などの専用コネクタ。最も高速かつ正確

▼

🌐 2. ブラウザ操作

Claude in Chrome 経由でブラウザを操作。コネクタが利用できない場合

▼

🖥️ 3. Computer Use (フォールバック)

デスクトップ上でクリック、タイピング、ナビゲーション。汎用性が高いが低速

利用シナリオ

🏢 社内ツールの操作

コネクタが存在しない社内ダッシュボードやデスクトップアプリケーションを Claude が直接操作。API 連携がなくても、画面操作で業務を自動化できます。

📱 UX テストの実施

電話シミュレータを開いて UX テストを実施。Claude が画面を操作してユーザーフローを検証し、結果をまとめます。

📊 複数ソースからのデータ収集

複数のアプリケーションやウェブサイトから情報を集め、スプレッドシートにデータを入力。競合分析や市場調査を効率化します。

📱 Dispatch でリモート作業

外出先からスマートフォンで「デスクトップのメールアプリで未読メールの要約を作成して」と指示。Claude が自動で実行し、結果を通知します。

開発者への影響

対象ユーザー

⭐ Pro / Max プラン

macOS の Claude Desktop アプリを利用しているユーザーが対象です。

🤖 Cowork ユーザー

Cowork 機能を活用して業務効率化を図っているユーザーに大きな改善です。

🏢 社内ツール連携

コネクタが存在しない社内ツールやデスクトップアプリを Claude と連携させたいユーザーに最適です。

必要なアクション

1

アプリの更新

macOS の Claude Desktop アプリを最新バージョンに更新してください。

2

プランの確認

Pro または Max プランに加入していることを確認してください。

3

Cowork でタスクを依頼

Cowork モードで Claude にタスクを依頼すると、必要に応じて Computer Use が自動的に使用されます。

4

アクセス許可の管理

各アプリケーションへのアクセス許可を求められた場合は、内容を確認して承認または拒否してください。

留意事項

🧪 研究プレビュー

本機能は研究プレビューとして提供されています。機能の仕様は今後変更される可能性があります。

⚠️ 画面操作は低速

画面操作は直接統合と比較して低速であり、複雑なタスクではリトライが必要になる場合があります。コネクタやブラウザ操作で対応可能なタスクはそれらが優先されます。

💻 macOS 限定

現時点では macOS のみの対応です。Windows 対応は近日公開予定です。
