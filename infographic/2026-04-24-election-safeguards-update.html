Anthropic が 2026 年選挙に向けた Claude の安全対策を公開

2026-04-24

概要: 6 つの柱

Anthropic は 2026 年の米国中間選挙やその他の主要選挙に向けた Claude の選挙安全対策に関する包括的なアップデートを公開しました。政治的バイアスの測定・防止、ポリシーの施行とテスト、自律的影響工作テスト、信頼性の高い選挙リソースの共有、最新情報の提供、第三者レビューの 6 つの柱で構成される取り組みが詳細に報告されています。

⚖

政治的バイアスの測定と防止

あらゆる政治的見解に対して同等の深さと敬意をもって対応。評価方法論とデータセットをオープンソースとして公開。

ポリシーの施行とテスト

Usage Policy に基づく選挙関連の禁止事項を、自動分類器と脅威インテリジェンスチームで監視。600 プロンプトを用いたテストで高い適切応答率を達成。

自律的影響工作テスト

Mythos Preview と Opus 4.7 を対象に初めて実施。安全装置の有無による自律的タスク遂行能力を評価。

選挙リソースの共有

選挙バナーを表示し、Democracy Works が運営する TurboVote にユーザーを誘導。ブラジル選挙向けにも展開予定。

最新情報の提供

Web 検索機能を活用して候補者情報、投票所の場所、選挙日程などをリアルタイムで提供。

6 つの柱: 主要機能

⚖ 政治的バイアス防止

左右の政治的見解に対する応答の深さ、敬意、バランスを定量的に測定。Opus 4.7 で 95%、Sonnet 4.6 で 96% の政治的中立性スコアを達成。評価方法論はオープンソースとして公開。

📋 ポリシー施行

自動分類器と脅威インテリジェンスチームによる監視体制。有害プロンプト 300 件 + 正当プロンプト 300 件の合計 600 プロンプトでテスト。適切な応答率は 99.8-100%。

🔎 自律影響工作テスト

Mythos Preview と Opus 4.7 で初実施。安全装置ありではほぼすべてのタスクを拒否。安全装置なしでは半数以上のタスクを完了可能であることを確認。

🗳 選挙リソース共有

選挙関連の質問に対して選挙バナーを表示し、TurboVote (Democracy Works) にユーザーを誘導。ブラジル選挙用にも展開予定。

🌐 最新情報提供

Web 検索機能を活用し、候補者情報、投票所の場所、有権者登録方法、選挙日程などをリアルタイムで提供。トリガー率は 92-95%。

🏛 第三者レビュー

The Future of Free Speech (Vanderbilt University)、Foundation for American Innovation、Collective Intelligence Project の 3 組織による独立した外部評価。

自律的影響工作テスト結果

Mythos Preview と Opus 4.7 を対象に初めて実施された自律的影響工作テスト。安全装置の有無でモデルの振る舞いが大きく異なることが確認されました。

安全装置あり

  • ほぼすべてのタスクを拒否
  • 影響工作の遂行を適切にブロック
  • 多層防御が有効に機能

安全装置なし

  • Mythos Preview: 半数以上のタスクを完了
  • Opus 4.7: 半数以上のタスクを完了
  • 継続的な警戒と改善が必要

ユースケース

Claude API を利用する開発者

選挙関連のコンテンツを扱うアプリケーションを構築する際、Claude の選挙安全対策が自動的に適用されます。Usage Policy の選挙関連禁止事項を確認し、アプリケーションの使用ケースが準拠していることを確認してください。

政治・メディア関連サービスの開発者

Claude の政治的中立性と有害コンテンツ拒否の特性を理解した上でアプリケーションを設計する必要があります。選挙バナー表示の動作を UI 設計に組み込むことが推奨されます。

多国籍サービスの開発者

ブラジルなど米国以外の選挙に対しても安全対策が拡大される点を考慮してください。Web 検索機能が有効になっていることを確認し、選挙関連の最新情報をユーザーに提供できるようにしましょう。

開発者向け必要なアクション

  • Usage Policy の確認: 選挙関連の禁止事項を理解し、アプリケーションの使用ケースが準拠していることを確認
  • 選挙バナー表示の考慮: Claude が選挙関連の質問に対してバナーを表示する動作を、アプリケーションの UI 設計に組み込む
  • Web 検索機能の有効化: 選挙関連の最新情報を提供するために、Web 検索機能が有効になっていることを確認
  • 政治的中立性の活用: Claude の政治的中立性を活かし、多様な政治的見解を持つユーザーに対して公平なサービスを提供
