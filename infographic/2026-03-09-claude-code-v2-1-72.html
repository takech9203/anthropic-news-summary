Claude Code v2.1.72 リリース:
トークンコスト最大 12 倍削減とバンドルサイズ 510KB 縮小

2026-03-09

v2.1.72 新機能 8 件 バグ修正 24+ 件

概要

💰

プロンプトキャッシュ修正で入力トークンコスト最大 12 倍削減

SDK の query() 呼び出しにおけるキャッシュ無効化を修正。キャッシュヒット率が大幅に向上し、特に大規模プロジェクトでの運用コストが劇的に低減されます。

⚡

Bash パーサーのネイティブモジュール化でメモリリーク解消

JavaScript ベースの Bash コマンドパーサーをネイティブモジュールに置き換え。メモリリークが解消され、初期化速度も向上しています。

📦

バンドルサイズ約 510KB 縮小

不要な依存関係の削除と最適化により、パッケージサイズが約 510KB 削減されました。

🆕

新機能 8 件追加

/copy w キー、/plan 説明引数、ExitWorktree ツール、エフォート簡素化、Bash 自動承認拡張、Agent model パラメータなど。

🔧

バグ修正 24 件以上

音声モード、プラグイン、権限ルール、並列ツール呼び出し、VS Code 統合など幅広い領域の安定性が改善されています。

Before / After

Before

  • SDK query() でプロンプトキャッシュが毎回無効化
  • 高い入力トークンコスト
  • JavaScript ベースのパーサーでメモリリーク発生
  • 長時間セッションで CPU 使用率が増大

After

  • キャッシュ正常動作でヒット率大幅向上
  • 入力トークンコスト最大 12 倍削減
  • ネイティブモジュールでメモリリーク解消
  • CPU 消費の最適化で安定動作

主要新機能

📋 /copy w キー

フォーカス中の選択範囲をクリップボードを介さずファイルに直接書き出し。SSH 環境で特に有用です。

📝 /plan 説明引数

/plan fix the auth bug のように、プラン作成時に説明を直接指定可能に。

🔀 ExitWorktree ツール

EnterWorktree セッションから離脱するための専用ツールを追加。ワークツリー管理がより柔軟に。

🎯 エフォート簡素化

max を廃止し、低 ○ / 中 ◐ / 高 ● の 3 段階に整理。直感的なレベル指定が可能に。

🔧 Bash 自動承認拡張

lsof、pgrep、tput、ss、fd、fdfind を自動承認リストに追加。確認プロンプトなしで実行可能に。

🤖 Agent model パラメータ

呼び出しごとのモデルオーバーライドが再度可能に。タスクに応じた最適なモデル選択ができます。

ユースケース

💻 SSH / リモート開発

/copy w キーでクリップボードを介さずファイルに直接書き出し。SSH 環境でのコピー操作が不要になり、リモート開発のワークフローが大幅に改善されます。

💰 コスト最適化

プロンプトキャッシュ修正により API コストが大幅に削減されます。特に大規模プロジェクトや頻繁に API を呼び出すワークフローで、運用コストの劇的な低減が期待できます。

⏰ 長時間開発セッション

CPU 使用率の最適化と Bash パーサーのメモリリーク解消により、長時間セッションでも安定した動作を維持できます。

まとめ

Claude Code v2.1.72 は、コスト効率とパフォーマンスの両面で大きな改善を実現したリリースです。最も注目すべきは、プロンプトキャッシュ無効化の修正による入力トークンコスト最大 12 倍の削減で、特に大規模なプロジェクトや頻繁に API を呼び出すワークフローにおいて、運用コストの大幅な低減が期待できます。Bash パーサーのネイティブモジュール化により、メモリリークの解消と初期化速度の向上が実現されたほか、バンドルサイズも 510KB 縮小されています。新機能 8 件とバグ修正 24 件以上により、全体的な安定性と利便性が大幅に向上しています。
