Claude Managed Agents: エフォートレベル、Webhook 拡張、セッションシーディング

2026-07-22

概要

⚡

エフォートレベル指定

エージェントのモデル設定にエフォートレベル (low/medium/high/xhigh/max) 指定可能に

🔔

Webhook ライフサイクル拡張

Webhook が Environment (4 イベント) と Memory Store (3 イベント) ライフサイクルをカバー

📨

セッションシーディング

セッション作成時に最大 50 個の初期イベントをシーディング可能

🔄

version フィールドのオプション化

version フィールドがオプション化 (楽観的並行性 or 無条件更新)

📡

スレッドイベントデルタ

スレッドレベルのイベントデルタでサブエージェントのリアルタイムプレビュー

アップデート一覧

⚡

エフォートレベル

low〜max の 5 段階でトークン消費と推論深度を制御

🔔

Webhook 拡張

environment.* (4 種) と memory_store.* (3 種) のライフサイクルイベント

📨

セッションシーディング

initial_events で 1 コール起動

🔄

version オプション化

CI/CD での宣言的管理を容易に

📡

スレッドイベントデルタ

サブエージェントのテキスト生成をリアルタイムプレビュー

エフォートレベル詳細

レベル 説明 推論深度
max トークン制約なし、最大能力
xhigh 長時間タスク向け拡張能力
high デフォルト、複雑な推論
medium バランス
low 最も効率的、サブエージェント向け

コードサンプル

Python エフォートレベル付きエージェント作成 
agent = client.beta.agents.create(
    name="Research Agent",
    model={"id": "claude-opus-4-8", "effort": "xhigh"},
    system="You are a thorough research agent.",
    tools=[{"type": "agent_toolset_20260401"}],
)
Python セッションシーディング 
session = client.beta.sessions.create(
    agent=agent.id,
    initial_events=[
        {"type": "user.message", "content": [{"type": "text", "text": "Analyze Q2 report."}]},
        {"type": "user.define_outcome", "rubric": "Include revenue growth and YoY comparisons."},
    ],
)
