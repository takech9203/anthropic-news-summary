Claude Code アップデート

Claude Code v2.1.198 — Chrome GA + バックグラウンドエージェント自動 PR

2026-07-02

概要

🌐

Claude in Chrome が一般提供開始

ベータ提供されていた Chrome 拡張機能が正式に GA 化。ブラウザから直接 Claude Code の機能にアクセスし、Web ページのコンテキストを活かした作業が可能に

🤖

バックグラウンドエージェントの自動 PR 作成

claude --bg で起動したエージェントがコード作業完了後に自動でコミット、プッシュ、ドラフト PR を作成。開発者の介入なしに PR まで完了

📊

新スキル /dataviz の追加

チャートやダッシュボードのデザインガイダンスを提供する新スキル。実行可能なカラーパレットバリデーターでデータ可視化の品質を事前検証

🧠

サブエージェント品質向上

拡張思考設定の継承、Explore エージェントのモデル継承 (Opus 上限)、チームメイトの障害報告と自動リトライにより委譲タスクの品質が向上

🌉

ゲートウェイ拡張と信頼性向上

Claude Platform on AWS 対応の追加、model-not-found 時のフェイルオーバー改善、ECONNRESET リトライ、STS トークン自動更新

Before / After

Before

  • バックグラウンドエージェントは作業完了後に停止し、確認を求める
  • 開発者が手動でコミット、プッシュ、PR 作成を実行
  • エージェント完了通知は手動確認が必要
  • サブエージェントは Haiku 固定で品質に制限

After

  • エージェントが自動でコミット → プッシュ → ドラフト PR 作成
  • Notification フックで完了/入力要求を自動通知
  • Explore エージェントがメインセッションのモデルを継承
  • 拡張思考設定もサブエージェントに継承

コードサンプル

Bash アップデートとバージョン確認 
# Claude Code を最新バージョンに更新
npm install -g @anthropic-ai/claude-code@latest

# バージョン確認
claude --version
# Claude Code v2.1.198
Bash バックグラウンドエージェントで自動 PR 
# バックグラウンドエージェントを起動 (完了時に自動で PR 作成)
claude --bg "auth モジュールのリファクタリングを行い、テストを追加してください"

# エージェントビューで状態を確認
claude agents
JSON Notification フックの設定例 
{
  "hooks": {
    "Notification": [
      {
        "matcher": "agent_completed",
        "command": "notify-send 'Claude Agent' 'タスクが完了しました'"
      },
      {
        "matcher": "agent_needs_input",
        "command": "notify-send 'Claude Agent' '入力が必要です'"
      }
    ]
  }
}
Bash /dataviz スキルの使用例 
# /dataviz スキルの使用例
claude
> /dataviz 売上データのダッシュボードを設計してください

ユースケース

非同期コードレビューの完全自動化

claude --bg でリファクタリングタスクを委譲し、エージェントが自動で PR を作成。Notification フックでデスクトップ通知を受け取り、完了後にレビューするだけのワークフローが実現。開発者は別タスクに集中できる

ブラウザからの即座なコード支援

Chrome 拡張機能 (GA) により、GitHub の PR レビュー画面やドキュメント閲覧中に Claude Code を直接呼び出し、Web ページのコンテキストを活かしたコード提案やバグ分析を即座に実行

エンタープライズ環境での安定運用

Claude Platform on AWS のゲートウェイ対応、STS トークン自動更新、ECONNRESET リトライ、フェイルオーバー改善により、AWS ベースのエンタープライズ環境での信頼性と可用性が大幅に向上

技術的な改善詳細

🔄 ECONNRESET リトライ

レスポンス中のネットワーク切断でターンが中断されていた問題を修正。バックオフ付きリトライに変更

🍎 macOS ローカルネットワーク

Local Network エンタイトルメント宣言により "no route to host" エラーを解消

🔑 STS トークン自動更新

AWS/Mantle セッションで STS トークン期限切れ時に awsAuthRefresh が自動実行

🎨 highlight.js 11 対応

コードブロック、差分、ファイルプレビューのシンタックスハイライト精度が向上

📝 /diff 自動更新

/diff パネルがブランチ切替やセッション外コミット時に自動更新されるように改善

🗑️ /agents ウィザード廃止

エージェント管理は Claude への指示または .claude/agents/ の直接編集に統一

必要なアクション

1

v2.1.198 にアップデート

npm install -g @anthropic-ai/claude-code@latest で最新バージョンに更新

2

Chrome 拡張機能のインストール

Chrome ウェブストアから Claude in Chrome 拡張機能をインストール

3

通知フックの設定

バックグラウンドエージェントの通知を受け取るために Notification フックを .claude/settings.json に設定

4

/agents ウィザード廃止への対応

エージェント管理を Claude への指示または .claude/agents/ ファイルの直接編集に移行

5

ゲートウェイ設定の確認

AWS 利用者は anthropicAws プロバイダーの追加を活用するかどうか確認
