新機能

🖥️ フリッカーフリーレンダリング

環境変数 CLAUDE_CODE_NO_FLICKER=1 を設定することで、代替スクリーンを使用したフリッカーフリーレンダリングと仮想スクロールバックが有効になります。

🪝 PermissionDenied フック

オートモードの分類器が拒否した後に発火するフック。{retry: true} を返すことでモデルにリトライを指示できます。

@ サブエージェントメンション補完

名前付きサブエージェントが @ メンションのタイプアヘッド候補に表示されるようになりました。

バグ修正 -- 29 件の内訳

🧠 メモリと安定性 6 件

  • 大きな JSON 入力が LRU キャッシュキーとして保持されるメモリリークを修正
  • 1 GiB 超のファイルに対する Edit ツール使用時の OOM クラッシュを修正
  • 50 MB 超のセッションファイルからメッセージを削除する際のクラッシュを修正
  • ツールスキーマのバイト変更に起因するプロンプトキャッシュミスを修正
  • ネストされた CLAUDE.md ファイルが長時間セッションで数十回再注入される問題を修正
  • StructuredOutput のスキーマキャッシュバグにより約 50% の失敗率が発生する問題を修正

💻 クロスプラットフォーム互換性 5 件

  • Windows で Edit/Write ツールが CRLF を二重化し、Markdown のハードラインブレイクを削除する問題を修正
  • Windows Terminal Preview 1.25 で Shift+Enter が改行ではなく送信になる問題を修正
  • Windows PowerShell 5.1 で git push などがエラーと誤報告される問題を修正
  • macOS Apple Silicon で音声モードがマイク権限をリクエストしない問題を修正
  • Windows の音声モードで "WebSocket upgrade rejected with HTTP 101" エラーが発生する問題を修正

🖥️ UI とレンダリング 8 件

  • 長時間セッションでスクロールアップ時にスクロールバックが消える問題を修正
  • 折り畳まれた検索/読み取りグループのバッジが並列ツール使用時に重複する問題を修正
  • /btw の長いレスポンスがクリップされスクロール不可だった問題を修正
  • デーヴァナーガリー文字やその他の結合文字テキストが切り詰められる問題を修正
  • メインスクリーンターミナルでのレイアウトシフト後のレンダリングアーティファクトを修正
  • iTerm2 + tmux 環境でのストリーミング中の周期的な UI ジッターを修正
  • 送信後にバックグラウンドメッセージ到着時にプロンプトが一時的に消える問題を修正
  • 通知の invalidates が現在表示中の通知を即座にクリアしない問題を修正

📂 セッションとデータ 10 件

  • --resume で古い CLI バージョンや中断された書き込みからのツール結果があるとクラッシュする問題を修正
  • "Rate limit reached" の誤表示を修正し、実際のエラーとアクションのヒントを表示するように改善
  • LSP サーバーがクラッシュ後にゾンビ状態になる問題を修正し、自動再起動するように改善
  • フック条件が複合コマンドや環境変数プレフィクス付きコマンドにマッチしない問題を修正
  • CJK や絵文字を含むプロンプト履歴が 4 KB 境界で消失する問題を修正
  • /stats が 30 日以上前の履歴データを失う問題を修正
  • /stats がサブエージェント/フォークの使用量を除外してトークン数を過少カウントする問題を修正
  • セッションを Ctrl+B でバックグラウンド化した際にタスク通知が失われる問題を修正
  • SDK エラー結果メッセージの is_error: true 設定が正しく行われない問題を修正
  • 音声モードのプッシュトゥトーク機能が一部の修飾キーコンボバインドで動作しない問題を修正

改善 -- 8 件

  • PowerShell ツールのプロンプトをバージョン別 (5.1 vs 7+) の構文ガイダンスで改善
  • 思考サマリーのデフォルト無効化: インタラクティブセッションで思考サマリーがデフォルトで生成されなくなりました
  • オートモードで拒否されたコマンドが通知表示され、/permissions の Recent タブに記録されるように変更
  • /env が PowerShell ツールのコマンドにも適用されるように変更
  • /usage で Pro および Enterprise プランの冗長な "Current week" バーを非表示に変更
  • 折り畳みツールサマリーで ls/tree/du の場合に "Listed N directories" と表示するように変更
  • 画像貼り付け時の末尾スペース挿入を削除
  • 空のプロンプトに !command を貼り付けた際に Bash モードに入るように変更

技術的な詳細

🧠 LRU キャッシュのメモリリーク修正

大きな JSON 入力がキャッシュキーとして保持され続けることで、セッションが長くなるほどメモリ使用量が増大する問題が解消されました。長時間セッションの安定性に直接影響する重要な修正です。

📋 プロンプトキャッシュミスの解消

ツールスキーマのバイト列がセッション中に変化することで、キャッシュが無効化されトークン使用量が増加する問題が修正されました。これによりトークン効率が改善されます。

📄 CLAUDE.md の再注入防止

ネストされた CLAUDE.md ファイルが繰り返し注入されることで、コンテキストウィンドウが圧迫される問題が解決されました。長時間セッションでのコンテキスト効率が向上します。

コードサンプル

Bash PermissionDenied フックの実装例 
#!/bin/bash
# my-permission-handler.sh
# PermissionDenied フックのサンプル実装

# フックからの入力を読み取る
input=$(cat)

# コマンド情報を取得
command=$(echo "$input" | jq -r '.command')

# 特定のコマンドパターンを許可
if echo "$command" | grep -q "^git push"; then
  echo '{"retry": true}'
else
  echo '{"retry": false}'
fi
JSON PermissionDenied フックの設定 
{
  "hooks": {
    "PermissionDenied": [
      {
        "command": "my-permission-handler",
        "description": "カスタム権限チェック"
      }
    ]
  }
}
Bash フリッカーフリーレンダリングの設定 
# .bashrc または .zshrc に追加
export CLAUDE_CODE_NO_FLICKER=1

# 設定後に Claude Code を起動
claude

⚠️ 思考サマリーのデフォルト変更

v2.1.88 から、インタラクティブセッションで思考サマリーがデフォルトで生成されなくなりました。従来の動作を維持する場合は、設定に以下を追加してください。

{
  "showThinkingSummaries": true
}
Bash アップデートコマンド 
# npm でのアップデート
npm update -g @anthropic-ai/claude-code

# 現在のバージョン確認
claude --version

開発者への影響

🕐 長時間セッションユーザー: メモリリーク修正の恩恵

LRU キャッシュのメモリリーク、CLAUDE.md の再注入、プロンプトキャッシュミスの修正により、長時間の開発セッションでも安定したパフォーマンスが期待できます。OOM クラッシュの防止も含め、セッション継続性が大幅に向上しています。

💻 Windows ユーザー: クロスプラットフォーム互換性の向上

CRLF 処理の修正、PowerShell 5.1 互換性の改善、Windows Terminal Preview での Shift+Enter 動作修正、音声モードの WebSocket エラー修正など、Windows 環境での使用体験が大幅に改善されています。

🤖 オートモードユーザー: PermissionDenied フックによる柔軟な制御

オートモードで拒否されたコマンドに対してカスタムロジックを実行し、条件付きでリトライを指示できるようになりました。拒否されたコマンドの通知と /permissions タブへの記録により、権限管理の可視性も向上しています。

🔧 フックシステム利用者: 複合コマンド対応の改善

フック条件が複合コマンドや環境変数プレフィクス付きコマンドに正しくマッチするようになり、PreToolUse/PostToolUse フックで file_path が絶対パスで提供されるようになりました。

🎤 音声モードユーザー: マルチプラットフォーム修正

macOS Apple Silicon でのマイク権限リクエスト、Windows での WebSocket エラー、プッシュトゥトーク機能の修飾キー対応が修正されています。
