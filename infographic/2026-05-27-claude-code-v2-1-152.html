コードサンプル

Bash コードレビュー自動修正 
# Code review with auto-fix
/code-review --fix

# Simplify (internally calls /code-review --fix)
/simplify
YAML スキルフロントマター: disallowed-tools 
---
name: security-audit
description: セキュリティ監査用スキル
disallowed-tools:
  - Bash
  - Write
---
JSON SessionStart フック設定 
{
  "hooks": {
    "SessionStart": [{
      "command": "install-team-skills.sh",
      "timeout": 10000
    }]
  }
}
Bash install-team-skills.sh - reloadSkills 出力例 
#!/bin/bash
# install-team-skills.sh
git clone --depth 1 https://github.com/team/claude-skills.git .claude/skills/team/ 2>/dev/null
echo '{"reloadSkills": true, "hookSpecificOutput": {"sessionTitle": "Team Session"}}'

ユースケース

📂 チームスキルの動的配布

SessionStart フックでチーム共有スキルを自動インストールし、reloadSkills: true で即座に利用可能に。再起動不要でチーム全体のスキルセットを統一できます

🔧 コードレビューワークフロー

/code-review --fix または /simplify でレビュー指摘を自動修正。手動修正の手間を省き、コード品質を効率的に向上できます

🔒 セキュリティ監査スキル

disallowed-tools フロントマターで、セキュリティ監査中に Bash や Write ツールを無効化。読み取り専用の安全な監査環境を構築

🪝 メッセージ表示のカスタマイズ

MessageDisplay フックで機密情報のマスキングや、社内ルールに基づく表示フィルタリングを実現。組織ポリシーに準拠した出力制御が可能
