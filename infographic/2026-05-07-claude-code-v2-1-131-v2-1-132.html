新機能・環境変数

🆔 CLAUDE_CODE_SESSION_ID

Bash ツールのサブプロセス環境に設定されるセッション ID。フックに渡される session_id と一致し、サブプロセスからセッションを識別可能に

🖥️ CLAUDE_CODE_DISABLE_ALTERNATE_SCREEN=1

フルスクリーンの代替画面レンダラーをオプトアウトし、ターミナルのネイティブスクロールバックに会話を保持

📋 「Pasting...」フッターヒント

Ctrl+V で画像を貼り付ける際、クリップボードから読み取り中であることを示すフッターヒントが表示

🧠 MCP メモリリーク

stdio MCP サーバーが非プロトコルデータを stdout に書き込むと、バッファが無制限に蓄積。v2.1.132 で非プロトコルデータを検知時にバッファを破棄

📝 セッション復元

絵文字のサロゲートペア分割による不正 UTF-16 を読み込み時にサニタイズし、--resume の信頼性を向上

🖥️ フルスクリーン復帰

SIGCONT シグナルハンドラーで画面全体を即座に再描画。スリープ復帰時のブランク画面を解消

🎮 JetBrains 互換性

IDE 2025.2 のターミナルエミュレータを識別し、スクロールイベントの解釈を適切に調整

コードサンプル

Bash 新しい環境変数の使用例 
# 代替画面レンダラーをオプトアウト
export CLAUDE_CODE_DISABLE_ALTERNATE_SCREEN=1
claude

# フックスクリプトでセッション ID を活用
#!/bin/bash
echo "Current session: $CLAUDE_CODE_SESSION_ID"
# セッション固有のログに記録
echo "$(date): action" >> "/tmp/claude-$CLAUDE_CODE_SESSION_ID.log"
Bash アップデート方法 
# npm でのアップデート
npm update -g @anthropic-ai/claude-code

# Homebrew でのアップデート
brew upgrade claude-code

# 現在のバージョン確認
claude --version
