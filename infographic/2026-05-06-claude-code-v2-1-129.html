新機能・改善: プラグインシステム

🔗 --plugin-url

URL から .zip プラグインアーカイブをダウンロードし、一時ディレクトリに展開してセッションスコープで読み込み。セッション終了時に自動クリーンアップ

🧪 experimental 配下へ移行

プラグインマニフェストの themes と monitors は "experimental" 配下に移動。トップレベル宣言は警告付きで引き続き動作

新機能・改善: 環境変数・設定

🖥️ FORCE_SYNC_OUTPUT

CLAUDE_CODE_FORCE_SYNC_OUTPUT=1 で同期出力を強制有効化。Emacs eat など自動検出が効かないターミナルで正しい表示を実現

📦 AUTO_UPDATE

CLAUDE_CODE_PACKAGE_MANAGER_AUTO_UPDATE で Homebrew/WinGet 環境のバックグラウンドアップグレードを有効化。完了後に再起動を促す

🌐 MODEL_DISCOVERY

CLAUDE_CODE_ENABLE_GATEWAY_MODEL_DISCOVERY=1 でゲートウェイモデルディスカバリーを明示的にオプトイン。v2.1.126-128 の自動有効から変更

新機能・改善: UI/UX とテレメトリ

🔍 Ctrl+R 履歴復帰

全プロジェクトのプロンプト履歴を横断検索するデフォルト動作に復帰。Ctrl+S でプロジェクト/セッション限定に絞り込み可能

🎯 スピナーティップス

3P デプロイメント環境で Anthropic ファーストパーティのサーフェスを参照するティップスが非表示に。UX の一貫性向上

⚙️ skillOverrides

off / user-invocable-only / name-only の 3 段階でスキルの表示・非表示を正しく制御可能に

📊 OTel PR カウント

🐛 ポリシー拒否メッセージ

API Request ID がエラーメッセージに含まれるようになり、サポート問い合わせ時のデバッグが容易に

技術的な詳細

📐

--plugin-url アーキテクチャ

指定 URL から .zip をダウンロード → 一時ディレクトリに展開 → セッションスコープで読み込み → セッション終了時にクリーンアップ。v2.1.128 の --plugin-dir ZIP 対応を拡張したもの

🕐

プロンプトキャッシュ TTL 修正

特定条件下で 1 時間 TTL が 5 分にダウングレードされていた。cache_creation コストの不要な増加が発生。v2.1.129 では TTL が正しく維持され、キャッシュヒット率が設計通りに動作

🔒

OAuth リフレッシュ競合の解決

スリープ復帰時に複数セッションが同時にトークンリフレッシュを試行 → 競合で有効トークンが無効化。ロック機構を導入し、一度に 1 セッションのみがリフレッシュを実行するように

🌐

Gateway モデルディスカバリーのオプトイン化

v2.1.126 で導入した自動ディスカバリーが一部環境で問題を起こしていたため、明示的な CLAUDE_CODE_ENABLE_GATEWAY_MODEL_DISCOVERY=1 設定が必要に変更

Bash v2.1.129 新機能の使用例 
# URL からプラグインを読み込み
claude --plugin-url https://example.com/plugins/my-plugin.zip

# 同期出力を強制有効化 (Emacs eat ターミナル向け)

# 自動アップデートを有効化 (Homebrew 環境)
export CLAUDE_CODE_PACKAGE_MANAGER_AUTO_UPDATE=1
claude

# Gateway モデルディスカバリーを有効化
export CLAUDE_CODE_ENABLE_GATEWAY_MODEL_DISCOVERY=1
claude

移行ガイド

⚠️ プラグインマニフェストの更新

themes と monitors を "experimental": { ... } 配下に移動してください。トップレベル宣言は動作しますが、claude plugin validate で警告が表示されます

⚠️ Gateway モデルディスカバリーの有効化

v2.1.126-2.1.128 で自動有効だった機能を使う場合、CLAUDE_CODE_ENABLE_GATEWAY_MODEL_DISCOVERY=1 の設定が必要です

⚠️ Ctrl+R 履歴検索の変更

デフォルトが全プロジェクト横断検索に戻りました。プロジェクト限定検索が必要な場合は Ctrl+S で切り替えてください
