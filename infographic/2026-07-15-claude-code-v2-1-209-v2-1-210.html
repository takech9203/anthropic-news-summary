改善項目

主な機能・修正

🔒 ワークツリー分離修正

サブエージェントがメインリポジトリに git 操作を実行できる問題を解消。エージェント専用ワークツリーの完全分離を実現

🛡️ インジェクション対策

Agent ツールの間接プロンプトインジェクション耐性を強化。外部コンテンツからの悪意ある命令を遮断

⏱️ 経過時間カウンター

ツール実行時間のリアルタイム表示機能を追加。長時間の処理でも進捗を可視化

🧠 Sonnet 5 分類器

auto モードの権限分類器がデフォルトで Sonnet 5 を使用。判定精度の向上により安全性と利便性を両立

🔓 worktree lock 自動解放

強制終了セッションの残留ロックを自動クリーンアップ。再起動時のロック競合問題を解消

♿ スクリーンリーダー対応

権限モード切り替えの音声通知に対応。視覚に頼らずに操作状態を把握可能に

Before / After

Before

  • ツール実行中は表示が固まって見える
  • エージェントワークツリーの git 操作がメインリポジトリに影響
  • 強制終了後にロックが残留し再起動に支障
  • 権限モード切り替えの視覚的確認のみ

After

  • リアルタイムカウンターで進捗を可視化
  • ワークツリー完全分離でセキュリティ確保
  • 残留ロックの自動クリーンアップ
  • スクリーンリーダーによる音声通知対応
