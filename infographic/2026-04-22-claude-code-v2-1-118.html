技術詳細

Vim ビジュアルモード

Vim エミュレーションに本格的なビジュアルモードが追加されました。

  • カーソル移動キー (h/j/k/l/w/b/e 等) による選択範囲の拡縮
  • オペレータ (d で削除、y でヤンク、c で変更) による操作
  • 選択範囲のビジュアルフィードバック (ハイライト表示)

MCP OAuth 信頼性改善の主要ポイント

  1. クロスプロセスロック: 複数プロセスの同時トークンリフレッシュのロック競合を解決
  2. macOS キーチェーン: 同時並行トークンリフレッシュの上書き問題を解消
  3. トークン有効期限: expires_in 省略や期限前取り消しへの耐性向上
  4. ステップアップ認可: スコープ不足の 403 レスポンスに対する適切な再同意プロンプト
  5. 認証情報永続化: Linux/Windows でのクラッシュとファイル破損を防止

auto モード $defaults 構文

従来の設定ではカスタムルールを設定するとビルトインリストが完全に置き換えられていました。"$defaults" をリストに含めることで、ビルトインリストを維持しつつカスタムルールを追加できます。
