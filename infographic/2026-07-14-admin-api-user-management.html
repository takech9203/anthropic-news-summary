概要

🏢

Enterprise 向けベータ公開

Claude Enterprise 組織向け Admin API ユーザー管理機能をベータ公開

👥

メンバー管理の API 化

メンバー一覧、ロール変更、招待管理をプログラムから操作

🏷️

グループと RBAC

グループ作成・管理と RBAC ベースのアクセス制御

カスタムロール

カスタムロールの読み取りと権限管理

監査スコープ

read:org_audit スコープで全 GET エンドポイントにアクセス可能

Before / After

Before

  • claude.ai 管理画面から手動でユーザー管理
  • 大規模組織で運用負荷大
  • オンボーディングに時間がかかる
  • 監査が手動作業

After

  • API から自動化可能
  • SCIM/SSO 統合
  • プログラムによるオンボーディング
  • 監査スコープで自動監査

5 つのリソース

👤 Members

一覧取得、メール検索、ロール変更、削除。組織メンバーの包括的な管理を API 経由で実現。

📨 Invites

招待作成、一覧取得、取り消し。プログラムからの一括招待で効率的なオンボーディング。

🏷️ Groups

グループ CRUD 操作。チームや部門単位でのアクセス制御を柔軟に構成。

👥 Group Members

メンバー追加・削除。グループへのメンバーシップ管理を自動化。

🔐 Custom Roles

ロールと権限の読み取り。RBAC ベースの細かなアクセス制御を実現。

コードサンプル

Bash メンバー一覧の取得 
# メンバー一覧の取得
curl "https://api.anthropic.com/v1/organizations/users?limit=20" \
  -H "x-api-key: $ANTHROPIC_ADMIN_KEY" \
  -H "anthropic-version: 2023-06-01"
Bash グループの作成 (ベータヘッダー必須) 
# グループの作成 (ベータヘッダー必須)
curl -X POST "https://api.anthropic.com/v1/organizations/rbac_groups" \
  -H "content-type: application/json" \
  -H "x-api-key: $ANTHROPIC_ADMIN_KEY" \
  -H "anthropic-beta: ce-user-management-2026-07-13" \
  -d '{"name": "Engineering"}'

ユースケース

🚀 自動オンボーディング

新入社員の招待からグループ割り当てまでを自動化。HR システムとの連携により、入社初日から適切なアクセス権を付与。

🔍 セキュリティ監査

read:org_audit スコープで組織全体のアクセス状況を監査。コンプライアンス要件への対応を自動化。

🔗 IdP 統合

SCIM/SSO と組み合わせた ID プロバイダー連携。既存の ID 管理基盤と Claude Enterprise をシームレスに統合。
