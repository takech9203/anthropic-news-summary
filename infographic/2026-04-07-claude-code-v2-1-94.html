コードサンプル

Bash Mantle 対応の有効化 
# 環境変数で Mantle を有効化
export CLAUDE_CODE_USE_MANTLE=1

# Claude Code を起動
claude
Bash エフォートレベルの調整 
# セッション内でエフォートレベルを確認・変更
/effort        # 現在のレベルを表示
/effort medium # 従来のデフォルトに戻す
/effort high   # 新しいデフォルト - より深い分析
/effort low    # 軽量モード
JSON UserPromptSubmit フックでセッションタイトルを設定 
{
  "hooks": {
    "UserPromptSubmit": [
      {
        "command": "echo '{\"sessionTitle\": \"My Custom Title\"}'",
        "type": "command"
      }
    ]
  }
}

移行ガイド

📈 デフォルトエフォートレベルの変更

対象: API キー、Bedrock/Vertex/Foundry、Team、Enterprise ユーザー

変更内容: デフォルトエフォートレベルが medium から high に変更されました。デフォルトでより多くのトークンを消費する可能性があります。

対応: 従来の動作を維持したい場合は /effort medium を実行するか、設定で固定してください。

🔌 プラグインスキル名の変更

対象: "skills": ["./"] でプラグインスキルを宣言しているプラグイン開発者

変更内容: スキルの呼び出し名がディレクトリのベースネームからフロントマターの name フィールドに変更されました。

対応: フロントマターの name フィールドが正しく設定されていることを確認してください。既存のスキル呼び出しが一致しなくなる可能性があります。
