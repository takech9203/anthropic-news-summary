技術的な詳細

🔄

MCP サーバー消失問題の原因

/clear 実行時にセッション状態がリセットされる際、MCP サーバーの接続情報が再初期化されないまま破棄されていた。修正後は /clear 後のセッション再構築時に .mcp.json、プラグイン、コネクタの設定を再ロード

🔐

OAuth 並行リフレッシュの直列化

複数 MCP サーバーのトークン有効期限が同時に到来した際、最初のリフレッシュ成功でリフレッシュトークン自体が無効化され後続が失敗。リフレッシュ操作を直列化し単一のトークンローテーションを共有する仕組みを導入

🧠

拡張思考リダクトブロック対応

ツールコール直後にリダクトされた思考ブロックが出力された場合、メッセージ構造が API スキーマに合致せず 400 エラー。リダクトブロックの位置を適切にハンドリングし API リクエスト構造を正規化
