Claude Code Changelog

Claude Code v2.1.187 リリース: サンドボックス資格情報保護、モデル制限、マウスサポートなど

2026-06-24

主要ポイント

🛡️

sandbox.credentials 設定

サンドボックスコマンドの資格情報ファイル・秘密環境変数の読み取りをブロック。セキュリティを強化し、機密情報の漏洩を防止。

🏢

組織レベルのモデル制限

管理者がモデルピッカー、--model、/model、ANTHROPIC_MODEL に対して制限を設定可能。コスト管理とセキュリティポリシーの強制を実現。

🖱️

マウスクリックサポート

フルスクリーンモードの選択メニューでマウスクリック操作が可能に。キーボードに加えてマウスでも操作できるように。

⏱️

MCP ツールタイムアウト

5 分間応答がないリモート MCP ツール呼び出しがエラーで中断。CLAUDE_CODE_MCP_TOOL_IDLE_TIMEOUT 環境変数で上書き可能。

🌳

サブエージェント深度追跡修正

再開・フォークされたサブエージェントの深度管理が改善。エージェントチェーンの安定性が向上。

コードサンプル

Bash Claude Code v2.1.187 使用例 
# Claude Code を最新バージョンにアップデート
claude /update

# MCP ツールのタイムアウトをカスタマイズ (例: 10分に設定)
export CLAUDE_CODE_MCP_TOOL_IDLE_TIMEOUT=600000

# 特定モデルを指定して起動 (組織制限がある場合はメッセージが表示される)
claude --model claude-sonnet-4-6

# バックグラウンドジョブの実行
claude --bg "テストを実行してください"

# 会話の再開 (修正済み)
claude --resume <conversation-id>

ユースケース

🏢 エンタープライズセキュリティ

sandbox.credentials でサンドボックス内の資格情報漏洩を防止。機密性の高い環境でも安全に Claude Code を活用できる。

📋 組織ガバナンス

管理者がモデル利用を制限し、コスト・セキュリティポリシーを強制。チーム全体で一貫したモデル利用ポリシーを実現。

🔗 MCP 連携の信頼性向上

タイムアウト導入でハングアップを防止し、自動リカバリを実現。リモート MCP ツールとの連携がより堅牢に。
