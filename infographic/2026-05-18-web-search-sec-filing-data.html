Web Search ツールで SEC ファイリングデータの強化

2026-05-18

概要

📊

SEC ファイリングデータがよりリッチに返却

Web Search ツールが SEC EDGAR データベースからの検索結果を構造化データとして返却するよう改善されました。10-K、10-Q、8-K などの主要なファイリング形式に対応しています。

🔍

金融リサーチエージェントの基盤強化

SEC ファイリングの一次情報源に直接アクセスし、信頼性の高い金融分析を自動化するエージェントの構築が容易になりました。

📈

決算分析ワークフローの改善

企業の決算情報を SEC ファイリングから直接取得し、正確な財務分析を引用付きで提供できるようになりました。

🔗

引用付きの一次情報源アクセス

検索結果には引用情報が付与され、分析結果のトレーサビリティと信頼性が向上しています。

⚡

API インターフェース変更不要

内部改善のため、既存の web_search_20260209 ツールをそのまま使用可能。コード変更なしで恩恵を受けられます。

Before / After 比較

Before

  • SEC ファイリング検索で基本的なテキスト結果のみ返却
  • 構造化されていないプレーンテキスト
  • 情報源の特定が困難
  • 手動での引用確認が必要

After

  • リッチな構造化データとして返却
  • 引用情報が自動付与
  • 一次情報源へのトレーサビリティ
  • ファイリング種別の識別が容易

主要機能

📄 SEC ファイリングデータの充実

10-K 年次報告書、10-Q 四半期報告書、8-K 臨時報告書など、主要な SEC ファイリング形式のデータをリッチに取得できます。

🔗 引用付きソース

検索結果に citation 情報が含まれ、ソースアトリビューション付きの信頼性の高いレスポンスを生成します。

⚡ 動的フィルタリング対応

web_search_20260209 ツールの allowed_domains パラメータで sec.gov に限定した検索が可能。精度の高い結果を取得できます。

🤖 金融エージェント向け最適化

金融リサーチエージェントが SEC ファイリングに基づいた正確な分析を自動で実行できるよう最適化されています。

コードサンプル

Python SEC ファイリング分析エージェント 
import anthropic

client = anthropic.Anthropic()

response = client.messages.create(
    model="claude-opus-4-7",
    max_tokens=4096,
    system="あなたは金融リサーチアナリストです。SEC ファイリングの一次情報源に基づいて、"
           "正確で引用付きの分析を提供してください。",
    messages=[
        {
            "role": "user",
            "content": "Apple Inc. の最新の 10-K ファイリングから、"
                       "売上高と営業利益の推移を分析してください。",
        }
    ],
    tools=[
        {
            "type": "web_search_20260209",
            "name": "web_search",
            "max_uses": 10,
            "allowed_domains": ["sec.gov", "edgar.sec.gov"],
        }
    ],
)

for block in response.content:
    if block.type == "text" and hasattr(block, "citations"):
        if block.citations:
            for citation in block.citations:
                print(f"出典: {citation.title}")
                print(f"URL: {citation.url}")

ユースケース

📊 投資リサーチ

企業の財務状況を SEC ファイリングから自動分析。10-K や 10-Q の財務データを引用付きで取得し、投資判断に必要な情報を効率的に収集できます。

🔍 デューデリジェンス

M&A 前の企業調査を引用付きで実施。対象企業の SEC ファイリングを網羅的に分析し、リスク要因や重要な開示事項を特定します。

📋 コンプライアンス

規制関連の情報を一次情報源から収集。SEC の最新ファイリング情報に基づき、コンプライアンス要件の確認や報告書の作成を支援します。
