新機能 (Added)

⏸️

defer パーミッション決定

PreToolUse フックに "defer" 決定を導入。ヘッドレスセッションでツール呼び出し時に一時停止し、-p --resume で再開してフックを再評価可能

✨

フリッカーフリーレンダリング

CLAUDE_CODE_NO_FLICKER=1 環境変数で仮想スクロールバックを使用したオルトスクリーンレンダリングに切り替え。画面のちらつきを抑制

🚫

PermissionDenied フック

Auto モードの分類器がツール実行を拒否した後に発火。{retry: true} を返すことでモデルにリトライを指示可能

👥

名前付きサブエージェント

@ メンションのタイプアヘッド候補にサブエージェント名が表示されるようになり、サブエージェントへのアクセスが容易に

🔌

MCP 接続のノンブロッキング化

MCP_CONNECTION_NONBLOCKING=true で接続待ちをスキップ。--mcp-config サーバー接続に 5 秒タイムアウトを設定

アップデート方法

以下のコマンドで最新バージョンに更新できます。

bash 
# npm でのアップデート
npm update -g @anthropic-ai/claude-code

# 現在のバージョン確認
claude --version

設定変更の確認

本リリースで確認が必要な設定項目です。

思考サマリー (showThinkingSummaries)

インタラクティブセッションでデフォルト無効化されました。復元するには settings.json に "showThinkingSummaries": true を追加してください

cleanupPeriodDays

値が 0 に設定されている場合、バリデーションエラーが発生します。1 以上の値に変更してください

Hooks の file_path

PreToolUse/PostToolUse フックが絶対パスを受け取るようになりました。相対パスに依存しているフック処理の見直しが必要な場合があります
