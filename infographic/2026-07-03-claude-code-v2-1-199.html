概要

💾

部分出力保持

ストリーミング中断やサブエージェント停止時に部分作業を破棄せず保持

🔄

リトライ強化

429 サーバーレートリミットの自動リトライ + RETRY_WATCHDOG で最大 300 回

🖥️

バックグラウンド安定化

Linux デーモン自己停止・macOS SSH 起動失敗・stop レースの修正

⚡

スキル複数ロード

/skill-a /skill-b do XYZ で最大 5 つ同時ロード

Before / After

Before

  • ストリーミング中断で出力全破棄
  • サブエージェントが無言で失敗
  • SSL エラーでリトライ消費後にヒント表示

After

  • 部分出力保持 + incomplete ラベル
  • エラーと部分作業を親に返却
  • SSL エラーで即座に修正ガイダンス表示

新機能・改善

🔄 サーバー 429 自動リトライ

サブスクライバー向け、一時的サーバーレートリミットを自動バックオフ

🐕 RETRY_WATCHDOG

非容量系エラーのリトライ上限を 300 に拡大、MAX_RETRIES の 15 制限を撤廃

⚡ スラッシュスキル複数ロード

/code-review /security-review 修正して のように最大 5 つ同時ロード

🏷️ PR リンク表示改善

claude agents で PR リンクが #N 表示に簡素化
