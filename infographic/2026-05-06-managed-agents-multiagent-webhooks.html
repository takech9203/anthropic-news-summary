概要

🤖

マルチエージェントセッション

コーディネーターエージェントが最大 20 の専門エージェントにタスクを委任し、最大 25 の同時スレッドで並列処理を実行可能に

🎯

Outcomes - 目標駆動型セッション

ルーブリックに基づく自動評価と反復改善で、エージェントの作業を「会話」から「目標達成」に昇格

📡

Webhooks イベント通知

セッションおよび Vault のライフサイクルイベントを HTTPS エンドポイントにプッシュ通知。ポーリング不要に

🔑

Vault バックグラウンドリフレッシュ

mcp_oauth クレデンシャルが自動的にトークンをリフレッシュ。長時間セッションの信頼性が向上

🔍

フィルタリング・ソート

セッションをステータスで、イベントをタイプや作成時刻でフィルタリング可能に

主要機能の詳細

🔀 並列化パターン

独立したサブタスクを同時に実行し、結果を統合。コードレビュー + テスト作成 + セキュリティ監査を並列実行

🎓 専門化パターン

ドメイン特化のシステムプロンプトとツールを持つエージェントにルーティング

⬆️ エスカレーション

複雑なサブタスクをより高性能なモデルに委託。Sonnet から Opus へのスマートな切り替え

📋 ルーブリック評価

マークダウン形式の評価基準を定義。独立したグレーダーが成果物を評価し、基準を満たすまで自動反復

🔏 署名付き通知

SDK の unwrap() ヘルパーで署名検証。安全なイベント処理を簡潔に実装

📂 共有コンテナ

すべてのエージェントが同じファイルシステムを共有。成果物の受け渡しがシームレス

コードサンプル

Python マルチエージェントセッションの構築 
import anthropic

client = anthropic.Anthropic()

# 専門エージェントを作成
reviewer = client.beta.agents.create(
    name="Code Reviewer",
    model="claude-sonnet-4-7",
    system="You review code for bugs and best practices.",
    tools=[{"type": "agent_toolset_20260401"}],
)

test_writer = client.beta.agents.create(
    name="Test Writer",
    model="claude-sonnet-4-7",
    system="You write comprehensive unit tests.",
    tools=[{"type": "agent_toolset_20260401"}],
)

# コーディネーターエージェントを作成
coordinator = client.beta.agents.create(
    name="Engineering Lead",
    model="claude-opus-4-7",
    system="Coordinate engineering work.",
    multiagent={
        "type": "coordinator",
        "agents": [
            {"type": "agent", "id": reviewer.id},
            {"type": "agent", "id": test_writer.id},
        ],
    },
)

# セッション作成・実行
session = client.beta.sessions.create(
    agent=coordinator.id,
    environment_id=environment.id,
)
Python Outcomes による目標駆動型セッション 
RUBRIC = """
# DCF Model Rubric
## Revenue Projections
- Uses historical revenue data from last 5 fiscal years
- Projects revenue for at least 5 years forward
## Output Quality
- All figures in a single .xlsx with labeled sheets
- Sensitivity analysis on WACC included
"""

client.beta.sessions.events.send(
    session_id=session.id,
    events=[{
        "type": "user.define_outcome",
        "description": "Build a DCF model for Costco",
        "rubric": {"type": "text", "content": RUBRIC},
        "max_iterations": 5,
    }],
)
Python Webhook の署名検証 
from flask import Flask, request
import anthropic

client = anthropic.Anthropic()
app = Flask(__name__)

@app.route("/webhook", methods=["POST"])
def webhook():
    try:
        event = client.beta.webhooks.unwrap(
            request.get_data(as_text=True),
            headers=dict(request.headers),
        )
    except Exception:
        return "invalid signature", 400

    match event.data.type:
        case "session.status_idled":
            session = client.beta.sessions.retrieve(event.data.id)
            notify_user(session)
        case "vault_credential.refresh_failed":
            handle_refresh_failure(event.data.id)

    return "", 204

ユースケース

🔍 コードレビューパイプライン

コーディネーターがコードレビュー、テスト作成、セキュリティ監査を並列に委任。各専門エージェントが独立して評価し、コーディネーターが結果を統合してレポートを生成

📊 財務分析の品質保証

Outcomes のルーブリックで DCF モデルの評価基準を定義。エージェントがモデルを構築し、グレーダーが基準を満たすまで自動的に修正を反復

⚡ 非同期ジョブモニタリング

Webhooks で長時間実行セッションの完了を通知。ポーリングなしでジョブ完了やエラーを即座に検知し、Slack や PagerDuty に連携
