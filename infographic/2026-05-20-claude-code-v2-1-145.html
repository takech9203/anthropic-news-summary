コードサンプル

Bash claude agents --json の活用例 
# 実行中の Claude セッションを JSON で取得
claude agents --json

# tmux ステータスバーでの活用例
claude agents --json | jq '[.[] | select(.status == "awaiting_input")] | length'

# セッションピッカーでの活用例
claude agents --json | jq -r '.[] | "\(.id)\t\(.status)\t\(.task)"' | fzf | cut -f1

import json

ユースケース

外部スクリプティング統合

claude agents --json とステータスライン JSON の GitHub 情報を組み合わせることで、tmux ステータスバーやカスタムセッションピッカーから Claude Code セッションを直接管理できます。入力待ちのセッション数をリアルタイムで表示し、作業効率を向上させます。

マルチエージェントデバッグ

OTEL トレーシングの agent_id / parent_agent_id 属性により、Jaeger や Grafana でサブエージェント間の呼び出し関係を正確に可視化できます。スパンの親子関係が正しくネストされるため、複雑なマルチエージェントワークフローのボトルネック特定が容易になります。

国際チームでの Agent Teams 活用

非 ASCII 文字のエンコーディング修正により、日本語や中国語などのエージェント名が正しく処理されます。グローバルに分散したチームで Agent Teams を利用する際の安定性が大幅に向上しました。
