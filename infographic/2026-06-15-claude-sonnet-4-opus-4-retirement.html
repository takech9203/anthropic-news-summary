Claude Sonnet 4 / Opus 4 の廃止

2026-06-15

概要

⚠️

即時停止

claude-sonnet-4-20250514 と claude-opus-4-20250514 は完全に廃止。API リクエストはエラーを返します。

🔄

移行必須

Sonnet 4.6 / Opus 4.8 への移行が必須。後継モデルは全性能指標で改善されています。

🔬

研究者アクセス

External Researcher Access Program で継続利用可能。研究目的での限定アクセスが維持されます。

📅

約 2 か月の猶予

4 月 14 日の非推奨化発表から計画通りの実施。十分な移行期間が提供されました。

Before / After

Before - Sonnet 4 / Opus 4

  • 200k コンテキストウィンドウ
  • Extended Thinking (手動 budget 設定)
  • Assistant Prefill サポート
  • claude-sonnet-4-20250514
  • claude-opus-4-20250514

After - Sonnet 4.6 / Opus 4.8

  • 1M コンテキストウィンドウ (標準)
  • Adaptive Thinking (自動最適化)
  • Prefill 非サポート
  • claude-sonnet-4-6
  • claude-opus-4-8

主要変更点

🚫 完全廃止

廃止モデルへの API リクエストはエラーを返します。既存のアプリケーションは即座に影響を受けます。

📖 1M コンテキスト

後継モデルは標準で 1M トークンのコンテキストウィンドウを提供。従来の 200k から 5 倍に拡大。

🧠 Adaptive Thinking

手動 budget 設定の Extended Thinking から、自動最適化される Adaptive Thinking へ移行。

⚡ パフォーマンス

後継モデルは全性能指標で改善。コーディング、推論、エージェント能力が大幅に向上。

影響を受けるユーザー

💻 本番 API を使用するアプリケーション開発者

claude-sonnet-4-20250514 または claude-opus-4-20250514 を直接指定しているアプリケーションは、即座にエラーが発生します。モデル ID を後継モデルに変更し、Adaptive Thinking パラメータを追加する必要があります。

☁️ Bedrock / Vertex AI 経由の利用者

Amazon Bedrock や Google Cloud Vertex AI を経由して廃止モデルを使用している場合も同様に影響を受けます。各クラウドプロバイダーのモデル ID も更新が必要です。

🔬 研究者 (External Researcher Access Program)

研究目的のユーザーは External Researcher Access Program を通じて廃止モデルへのアクセスを継続できます。プログラムへの参加申請が必要です。
