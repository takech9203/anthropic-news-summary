v2.1.218 - コードレビューのバックグラウンド実行とアクセシビリティ強化

2026-07-22

v2.1.218

概要

🔍

/code-review がバックグラウンドサブエージェントとして実行

会話を占有せず、レビュー中も他の作業を継続可能に

♿

スクリーンリーダーでの削除テキスト読み上げ対応

単語削除・行削除時にスクリーンリーダーが変更を適切に通知

⚡

auto モードの権限ダイアログ削減

分類器に委譲することで、不要な権限確認を減らしスムーズな操作を実現

🌐

MCP 接続失敗時の HTTP ステータス/エラー表示

接続失敗の原因を即座に特定できるよう、HTTP ステータスコードとエラーメッセージを表示

🐛

Windows パス CJK 文字化け、左矢印会話消失など重大バグ修正

Windows 環境での CJK 文字を含むパス処理、左矢印キーによる会話消失、複数行ペーストの問題を修正

Before / After

Before

  • /code-review が会話全体を占有
  • レビュー中は他の作業が不可
  • レビュー完了まで待機が必要

After

  • バックグラウンドサブエージェントで実行
  • レビュー中も作業を継続可能
  • 完了時に通知で結果を受け取り

機能カテゴリ

🔍 コードレビュー改善

バックグラウンドサブエージェント実行により、会話のブロックを解消。レビュー結果は完了時に通知される。

♿ アクセシビリティ

削除テキスト読み上げ対応とフォーカス追跡の改善。スクリーンリーダー利用者の操作体験を向上。

⚡ auto モード改善

権限ダイアログ削減と分類器活用により、自動実行フローがよりスムーズに。

🐛 重要バグ修正

Windows CJK 文字化け、左矢印キーによる会話消失、複数行ペーストの問題を修正。

動作変更

  • 🔬 /deep-research は手動起動のみ: Claude が自発的に起動しなくなった。ユーザーが明示的にコマンドを実行する必要がある。
  • 🏷️ agent 名のコロン : 禁止: プラグイン名前空間の予約のため、エージェント名にコロンを含めることができなくなった。
  • 🔄 context: fork スキルがデフォルトでバックグラウンド実行: fork コンテキストを持つスキルは自動的にバックグラウンドで実行される。
