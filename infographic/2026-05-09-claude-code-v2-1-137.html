Claude Code Hotfix

Claude Code v2.1.137: Windows VS Code 拡張機能アクティベーション修正

2026-05-09

概要

🪟

Windows 環境のみに影響するホットフィックス

VS Code 拡張機能が Windows 環境でアクティベーションに失敗する問題を修正。前日リリースの v2.1.136 で発生したリグレッションへの迅速な対応です

⚡

迅速なリリース対応

v2.1.136 (52 件の変更を含む大規模リリース) の翌日にホットフィックスをリリース。問題の特定から修正リリースまで 24 時間以内の対応

影響範囲

⚠️

Windows + VS Code ユーザーのみ

macOS / Linux ユーザーには影響ありません。v2.1.136 で正常に動作している場合、アップデートは任意です。Windows で VS Code の Claude Code 拡張機能が動作しない場合のみ、v2.1.137 へのアップデートが必要です。

Before / After

Before (v2.1.136 on Windows)

VS Code 拡張機能がアクティベーションに失敗し、全機能が利用不可。コマンドパレット、インラインアシスト、サイドバー UI すべてが動作しない状態

After (v2.1.137)

Windows 環境で VS Code 拡張機能が正常にアクティベーション。全機能が期待通りに動作し、v2.1.136 の新機能も利用可能に

アップデート方法

Bash CLI アップデート手順 
# CLI のアップデート
npm update -g @anthropic-ai/claude-code

# バージョン確認
claude --version
# Expected: 2.1.137

📦 VS Code 拡張機能の更新

VS Code マーケットプレイスから自動的にアップデートされます。手動更新の場合は、拡張機能パネルから Claude Code を検索し最新バージョンに更新してください

対象ユーザー

🪟 Windows + VS Code ユーザー

v2.1.136 にアップデート後に拡張機能が動作しなくなった場合、v2.1.137 で解決されます。速やかにアップデートを推奨

🍎🐧 macOS / Linux ユーザー

本リリースによる影響はありません。v2.1.136 で正常に動作している場合、アップデートは任意です
