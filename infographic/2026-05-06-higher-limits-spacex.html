主な変更点

⚡ レートリミット倍増

Pro、Max、Team、シートベース Enterprise の全プランで Claude Code の 5 時間あたりレートリミットを 2 倍に。

🕐 ピーク時間制限撤廃

Pro および Max アカウントのピーク時間中のリミット削減を完全に廃止。いつでも同じ上限で利用可能。

🤖 Opus API 強化

Claude Opus モデルの API レートリミットを大幅に引き上げ。並列処理や高頻度リクエストに対応。

🚀 SpaceX Colossus 1

300MW 超の新規計算能力を 1 か月以内に確保。Pro/Max サブスクライバーの容量を直接改善。

🛰️ 軌道 AI コンピュート

複数ギガワット規模の軌道上 AI コンピュート容量の開発にも関心を表明。将来の拡張を示唆。

🌐 グローバル展開

アジア・ヨーロッパでの追加推論インフラ。規制産業向けのリージョン内対応を強化。

コードサンプル

Python 引き上げ後のレートリミットで高頻度リクエスト 
import anthropic

# Claude Opus モデルのレートリミットが大幅に引き上げられたため、
# より高頻度のリクエストが可能に
client = anthropic.Anthropic()

# 以前はレートリミットにより制限されていた並列リクエストが
# より多く処理可能に
response = client.messages.create(
    model="claude-opus-4-6-20260416",
    max_tokens=4096,
    messages=[
        {"role": "user", "content": "大規模なコード分析を実行してください"}
    ]
)

開発者への影響

🧑‍💻 Claude Code ヘビーユーザー

レートリミット倍増とピーク時間制限撤廃により、時間帯を気にせず集中的な開発作業が可能に。ワークフローの見直しで生産性向上が期待できる。

🔌 API 開発者 (Opus モデル利用)

大幅に引き上げられたレートリミットにより、並列リクエストやバッチ処理がスムーズに。スロットリングロジックの緩和を検討可能。

🏢 エンタープライズ顧客

シートベース Enterprise プランでもレートリミット倍増の恩恵を受けられる。国際展開により、リージョン内でのインフラ利用も拡充。

国際展開

🏛️

規制産業向けリージョン内インフラ

規制の厳しい産業の顧客向けに、各地域内でのインフラストラクチャを整備。データ主権要件に対応。

アジア・ヨーロッパでの推論拡充

Amazon との協力により、アジアおよびヨーロッパでの追加推論インフラを提供。レイテンシの改善とローカル対応を強化。

電力コストへのコミットメント

データセンターに起因する消費者電力価格の上昇分をカバーすることを約束。民主主義国との安全なサプライチェーンパートナーシップを推進。
