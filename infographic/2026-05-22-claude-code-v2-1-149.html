セキュリティ修正 (重大)

🚨

PowerShell パーミッションバイパス

cd../cd\/cd~/X: でワークスペース外アクセスが可能だった脆弱性を修正。全 cd 変種を検知しトラッキング

🚨

Git ワークツリーサンドボックス

ワークツリーからメインリポジトリ全体に書き込み許可されていた問題を修正。共有 .git ディレクトリのみに制限

🚨

PowerShell ワイルドカードルール

ネイティブ実行ファイルが未承認で実行可能だった問題を修正

🚨

変数トラッキングギャップ

PWD/OLDPWD/DIRSTACK の stale 値により正確なディレクトリ追跡ができていなかった問題を修正

Before / After

PowerShell パーミッション検知

Before

PowerShell の cd 関数がパーミッション検知をバイパスし、ワークスペース外にアクセス可能

After

全 cd 変種 (cd..、cd\、cd~、X:) を検知しトラッキング。ワークスペース外アクセスを確実にブロック

Git ワークツリーサンドボックス

Before

ワークツリーからメインリポジトリ全体に書き込み可能。意図しないファイル変更のリスク

After

共有 .git ディレクトリのみに制限。ワークツリーのサンドボックスが適切に機能
