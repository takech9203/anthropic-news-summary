ポイント

主な変更点

🏷️ Manual モード統一

CLI、VS Code、JetBrains 全環境でパーミッションモード "default" を "Manual" に名称変更。互換性のため旧名称も引き続き利用可能。

⏸️ AskUserQuestion 変更

ダイアログのデフォルト自動継続を無効化。アイドルタイムアウトによる自動継続は /config から明示的にオプトイン。

🧠 Sonnet 5 最適化

v2.1.201 で会話途中のシステムロールによるハーネスリマインダーを廃止。コンテキストウィンドウの無駄を削減。

🔄 デーモンハンドオーバー

ビルドタイムスタンプによるバージョン判定を導入。再インストールされた古いビルドによるデーモン奪取を防止。

♿ スクリーンリーダー対応

装飾グリフの非表示化、トランスクリプトシンボルの短ラベル読み上げ、テーブルの改善された読み上げ形式。

🖥️ tmux 互換性

tmux 3.4 以上で発生していたレンダリングフリッカーを修正。同期化ターミナル出力を有効化。

バックグラウンドエージェント修正一覧

  • スリープ/復帰後やストールしたセッション再開時の無言停止
  • Esc キャンセル後のストール復帰時にターンが再実行される問題
  • クラッシュ後の古い daemon.lock の PID 再利用によるエージェント再起動不能
  • 再インストールされた古いビルドによるデーモン奪取
  • ロスターの一時破損によるオーファンクリーンアップの永久無効化
  • 古いバイナリが新しいバージョンのフィールドを保持しない問題
  • デーモン再起動時のソケット認証トークン削除
  • バックグラウンド出力の制御バイトがターミナルに表示される問題

開発者への影響
