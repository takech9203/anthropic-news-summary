新機能

📊 waitingFor in --json

セッションのブロック理由を可視化。claude agents --json の出力にセッションが何を待っているかを示すフィールドが追加されました。

🔧 --tools Grep/Glob

ネイティブビルドで専用検索ツール提供。--tools フラグで Grep や Glob を明示的に指定して高速な検索が可能です。

⚙️ /effort 永続化確認

デフォルト設定の保存を明示。/effort コマンドで設定変更時に永続化するかどうかの確認が表示されます。

🖥️ Remote Control フッター

セッションリンク付き永続ピル表示。リモートセッション接続用のリンクが常時フッターに表示されます。

✏️ スラッシュコマンド改善

クリックで入力、Enter で実行の 2 段階方式。誤操作を防止し、コマンドの引数をプレビューできます。

📝 Devin Desktop リネーム

Windsurf からの名称変更に追従。Devin Desktop との連携設定が最新の名称に更新されました。

バグ修正カテゴリ

🛠️ 起動/設定

読み取り専用設定ディレクトリでのハング修正。ファイルシステムの権限に起因する起動失敗が解消されました。

🔒 セキュリティ

WebFetch パーミッション優先順位の修正、Windows パス正規化によるバックスラッシュ・大文字小文字の不一致を解消。

🔌 MCP

タイムアウト 1000ms 未満の正規化、LSP workspaceSymbol 修正。短いタイムアウト値が適切にハンドリングされます。

💻 agents ダッシュボード

8 件の修正を実施。表示幅、セッションアタッチ、ペースト操作、キューイング処理などの問題が改善されました。

Bash v2.1.162 新機能の使用例
