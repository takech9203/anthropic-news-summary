## 概要

Claude Code v2.1.123 が 2026 年 4 月 29 日にリリースされました。本リリースは、`CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1` を設定している環境で OAuth 認証が 401 リトライループに陥る問題を修正するホットフィックスです。前バージョン v2.1.122 (2026 年 4 月 28 日リリース) で導入された多数の新機能・改善に対する安定性修正となります。

## 詳細

### 背景

Claude Code は Anthropic が提供する CLI ベースの AI 開発支援ツールです。環境変数 `CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1` は、実験的な機能を無効化するためのオプションです。この設定を有効にしているユーザーにおいて、OAuth 認証フローが正常に完了せず、401 (Unauthorized) レスポンスに対するリトライが無限ループする問題が発生していました。

v2.1.123 はこの 1 件のバグ修正のみを含むホットフィックスリリースです。

### 主な変更点

#### バグ修正 - 1 件

- **OAuth 認証の 401 リトライループ修正**: `CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1` を設定している場合に、OAuth 認証が 401 エラーを繰り返しリトライし続ける問題が修正されました。実験的ベータ機能を無効化した状態でも、認証フローが正常に完了するようになりました

### 技術的な詳細

`CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1` 環境変数は、Claude Code の実験的なベータ機能を無効化するために使用されます。この設定が有効な場合、認証リクエストに含まれるべきパラメータの一部が欠落し、サーバー側で 401 レスポンスが返されていたと推測されます。Claude Code のリトライロジックが 401 を再試行可能なエラーとして処理したため、無限ループが発生していました。

本修正により、実験的ベータを無効化した状態でも OAuth 認証に必要なパラメータが正しく送信され、認証が正常に完了するようになりました。

## 開発者への影響

### 対象

- **`CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1` を設定しているユーザー**: この環境変数を使用して実験的機能を無効化している場合に直接影響を受けます。本修正により OAuth 認証が正常に機能するようになります
- **OAuth 認証を使用している全てのユーザー**: 認証のリトライロジックが改善されるため、将来的な認証エラー時の安定性が向上します

### 必要なアクション

以下のコマンドで最新バージョンに更新できます。

```bash
# npm でのアップデート
npm update -g @anthropic-ai/claude-code

# Homebrew でのアップデート
brew upgrade claude-code

# 現在のバージョン確認
claude --version
```

`CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1` を設定していて認証に問題が発生している場合は、速やかにアップデートしてください。

## コード例

```bash
# 実験的ベータを無効化した状態で Claude Code を使用 (v2.1.123 で修正済み)

claude

```

## まとめ

Claude Code v2.1.123 は、OAuth 認証の 401 リトライループを修正する 1 件のバグ修正のみを含むホットフィックスリリースです。`CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1` 環境変数を設定しているユーザーが認証できない問題が解消されました。

この問題は実験的ベータ機能を意図的に無効化しているユーザーにのみ影響するため、影響範囲は限定的です。ただし、該当する環境設定を使用している場合は速やかなアップデートを推奨します。その他のユーザーについては、前バージョン v2.1.122 の機能・修正が安定して動作しており、通常のアップデートサイクルでの更新で問題ありません。
