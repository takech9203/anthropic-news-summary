## 概要

Claude Code v2.1.137 が 2026 年 5 月 9 日にリリースされました。本リリースは、前日 (5 月 8 日) にリリースされた v2.1.136 で発生した Windows 環境での VS Code 拡張機能のアクティベーション失敗を修正するホットフィックスです。変更点は 1 件のみで、Windows ユーザーが VS Code で Claude Code 拡張機能を使用できない問題を解消します。

## 詳細

### 背景

v2.1.136 は 52 件の変更を含む大規模リリースでした。MCP 関連の修正、UI の改善、新しいエンタープライズ機能など多岐にわたる変更が含まれていましたが、その中でリグレッションが発生し、Windows 環境で VS Code 拡張機能がアクティベーションに失敗する問題が生じました。

Claude Code の VS Code 拡張機能は、エディタ内から直接 Claude Code を利用するための重要なインターフェースです。アクティベーションが失敗すると拡張機能が全く機能しないため、Windows ユーザーにとっては致命的な問題でした。Anthropic はこの問題を迅速に特定し、翌日にはホットフィックスとして v2.1.137 をリリースしました。

### 主な変更点

#### VS Code 拡張機能修正 - 1 件

1. **[VSCode] Windows でのアクティベーション失敗を修正**: VS Code 拡張機能が Windows 環境でアクティベーションに失敗する問題が修正されました

### 技術的な詳細

VS Code 拡張機能のアクティベーションは、拡張機能が読み込まれる際に実行される初期化プロセスです。このプロセスが失敗すると、拡張機能のすべての機能 (コマンドパレットからの Claude Code 起動、インラインアシスト、サイドバー UI など) が利用できなくなります。

v2.1.136 の大規模な変更の中で、Windows 固有のパス処理やプラットフォーム依存のロジックに影響を与える変更が含まれていた可能性があり、それが Windows 環境でのみアクティベーション失敗を引き起こしていました。v2.1.137 ではこの問題を修正し、Windows 環境で正常に拡張機能がアクティベーションされるようになりました。

## 開発者への影響

### 対象

- **Windows 環境で VS Code を使用している Claude Code ユーザー**: 本修正の直接的な対象です。v2.1.136 にアップデート後に拡張機能が動作しなくなった場合、v2.1.137 で解決されます
- **macOS / Linux ユーザー**: 本リリースによる影響はありません。既に v2.1.136 で正常に動作している場合、アップデートは任意です

### 必要なアクション

Windows 環境で VS Code の Claude Code 拡張機能が動作しない場合、以下の手順でアップデートしてください。

```bash
# CLI のアップデート
npm update -g @anthropic-ai/claude-code

# バージョン確認
claude --version
# Expected: 2.1.137
```

VS Code 拡張機能は、VS Code のマーケットプレイスから自動的にアップデートされます。手動で更新する場合は、VS Code の拡張機能パネルから Claude Code 拡張機能を検索し、最新バージョンに更新してください。

### 移行ガイド (該当する場合)

本リリースはホットフィックスのため、移行作業は不要です。v2.1.136 以前から直接 v2.1.137 にアップデートする場合も、特別な手順は必要ありません。

## まとめ

Claude Code v2.1.137 は、Windows 環境での VS Code 拡張機能のアクティベーション失敗を修正する単一のバグ修正を含むホットフィックスリリースです。

前日にリリースされた v2.1.136 は 52 件の変更を含む大規模アップデートでしたが、その中で Windows 固有のリグレッションが発生しました。Anthropic は迅速に対応し、翌日には修正版をリリースしています。この素早い対応は、Claude Code のリリースプロセスにおける品質管理とユーザー影響の最小化への取り組みを示しています。

Windows で VS Code を使用している Claude Code ユーザーは、速やかに v2.1.137 へのアップデートを推奨します。macOS や Linux ユーザーには影響のない変更であるため、アップデートは任意です。
