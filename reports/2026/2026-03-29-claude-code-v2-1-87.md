## 概要

Claude Code v2.1.87 が 2026 年 3 月 29 日にリリースされました。本リリースは、Cowork Dispatch でメッセージが配信されない問題を修正するホットフィックスです。変更点は 1 件のみの小規模リリースとなっています。

## 詳細

### 背景

Claude Code は Anthropic が提供する CLI ベースの AI 開発支援ツールです。v2.1.87 は v2.1.86 から 1 日後のリリースであり、Cowork Dispatch 機能に影響するバグの緊急修正を目的としたホットフィックスリリースです。

Cowork Dispatch は 2026 年 3 月 23 日に発表された機能で、複数の並列エージェントにタスクをディスパッチして同時実行できる仕組みです。この機能により、大規模なタスクを複数のエージェントに分割して効率的に処理できるようになりました。

### 主な変更点

#### バグ修正

- **Cowork Dispatch のメッセージ配信修正**: Cowork Dispatch でディスパッチされたエージェント間のメッセージが正しく配信されない問題を修正しました。この問題により、並列エージェント間の通信が失敗し、タスクの実行結果が正しく返されない状況が発生していました

### 技術的な詳細

Cowork Dispatch では、親エージェントが複数の子エージェントにタスクをディスパッチし、各エージェントが並列で処理を行います。処理結果は子エージェントから親エージェントへメッセージとして返されます。v2.1.87 で修正されたバグは、このメッセージ配信プロセスに影響するものであり、ディスパッチされたエージェントからのメッセージが親エージェントに到達しないケースがありました。

この問題はタスクの失敗やタイムアウトとして現れる可能性があり、特に複数のエージェントを同時にディスパッチする複雑なワークフローで影響が大きくなります。

## 開発者への影響

### 対象

- Cowork Dispatch 機能を利用して複数の並列エージェントにタスクをディスパッチしているユーザー
- Claude Code CLI を日常的に利用している全ての開発者

### 必要なアクション

以下のコマンドで最新バージョンに更新できます。

```bash
# npm でのアップデート
npm update -g @anthropic-ai/claude-code

# 現在のバージョン確認
claude --version
```

Cowork Dispatch を利用しているユーザーは、メッセージ配信の信頼性が向上するため、早急なアップデートを推奨します。

## まとめ

Claude Code v2.1.87 は、Cowork Dispatch でメッセージが配信されない問題を修正するホットフィックスリリースです。Cowork Dispatch は 2026 年 3 月 23 日に発表された比較的新しい機能であり、複数の並列エージェントにタスクをディスパッチして効率的に処理する仕組みです。

本バグにより、ディスパッチされたエージェント間のメッセージ通信が失敗するケースがありました。Cowork Dispatch を活用しているユーザーにとっては機能の信頼性に直結する重要な修正であり、該当ユーザーには早急なアップデートを推奨します。変更点は 1 件のみですが、並列エージェント間の通信基盤に関わる修正であるため、Dispatch 機能の安定運用に不可欠なリリースです。
